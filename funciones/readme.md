**Conceptos Básicos de Clases**

Una clase es como un plano o molde para crear objetos.
Define las propiedades (datos) y los comportamientos (funciones) que tendrán los objetos creados a partir de ella. Un objeto es una instancia de una clase.
Una clase se define con la palabra clave **class**.

**Constructores y Métodos**
2. Constructor (__init__)
El constructor es un método especial llamado __init__. Se ejecuta automáticamente cuando se crea un nuevo objeto de la clase.
self: Es el primer parámetro de cualquier método de instancia en Python, y siempre hace referencia a la propia instancia del objeto que se está creando o utilizando.
Inicialización: Se usa para inicializar los atributos (propiedades específicas) del objeto.

**4. Métodos**
Un método es simplemente una función definida dentro de una clase. Define las acciones o comportamientos que el objeto puede realizar.

**5. Principios de la POO**
Los dos pilares fundamentales de la POO que mencionaste son la Herencia y el Polimorfismo.
**Herencia**
La Herencia permite que una nueva clase (clase hija o subclase) adquiera los atributos y métodos de una clase existente (clase padre o superclase). Esto fomenta la reutilización de código.

**6. Polimorfismo**
Polimorfismo significa "muchas formas". En POO, se refiere a la capacidad de diferentes clases para responder a un mismo método de manera diferente.
Esto se ve claramente cuando una clase hija sobrescribe (redefine) un método de su clase padre.
