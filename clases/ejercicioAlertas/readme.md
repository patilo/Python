**Ejercicio de alertas**
en este ejercicio vamos a aplicar todos los conocimientos de clases, poo, metodos y constructores
crearemos una clase alerta con los siguientes metodos:
- constructor: que reciba el nivel de alerta
- mostrar_alerta: que muestre la alerta
- cambiar_nivel: que cambie el nivel de alerta
- menu: que muestre un menu con las opciones de mostrar alerta y cambiar nivel
- salir: que muestre un mensaje de despedida
