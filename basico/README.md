**🐍 Python Básico: Fundamentos Esenciales**

Este repositorio sirve como una guía rápida y un conjunto de ejemplos básicos para entender la sintaxis y las estructuras de datos fundamentales de Python. Ideal para principiantes.

**Tipos de Datos Primitivos**
Python es de tipado dinámico, lo que significa que no necesitas declarar el tipo de una variable. Aquí están los tipos de datos más comunes:

**Estructuras Condicionales (if, elif, else)**
Permiten que el programa tome decisiones basadas en si una condición es verdadera (True) o falsa (False).

**match case**
se utiliza para comparar una variable con varios valores, esto es muy util para los menus
se utilizan con valores fijos, no con variables, pueden ser numeros o texto

**Estructuras de Bucle (for y while)**
Se utilizan para repetir un bloque de código.
**Bucle for** (Iteración Definida)
Recorre una secuencia de elementos (listas, rangos, etc.).
**Bucle while** (Iteración Condicional)
Se ejecuta mientras la condición especificada sea verdadera.

**Estructuras de Datos Colecciones**
Python ofrece estructuras de datos integradas para almacenar colecciones de elementos.
**Listas (list)**
Colección ordenada y mutable (se puede modificar).
Se definen con corchetes [].
**Tuplas (tuple)**
Colección ordenada e inmutable (NO se puede modificar después de la creación).
Se definen con paréntesis ().
**Diccionarios (dict)**
Colección no ordenada (hasta Python 3.7+), mutable que almacena datos como pares Clave:Valor.
Se definen con llaves {}.

**funciones matematicas**
sum(): suma los elementos de una lista
max(): encuentra el numero mayor de una lista
min(): encuentra el numero menor de una lista
len(): cuenta la cantidad de elementos de una lista
range(): crea una secuencia de numeros
round(): redondea un numero
abs(): valor absoluto de un numero
pow(): potencia de un numero
sqrt(): raiz cuadrada de un numero
